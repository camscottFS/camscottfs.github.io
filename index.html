About Me

Seemless HTML5 Video

What I hope to practice and/or learn

After researching and viewing websites that have this I found it very useful and interesting. Inserting HTML5 Video and styling it with CSS to make it "seemless" could make the webpage look a lot better rather than using some sort of stationary image. I hope to learn ways to make the video appear as if it's not a video. An example of an HTML5 Video that was styled with CSS to appear seemless can be viewed here.
