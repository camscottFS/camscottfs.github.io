Responsive Navigation Bar

What I hope to practice and/or learn

Responsive Navigation is important now-a-days because a lot of people view websites from their mobile device as well as their desktop/laptop. It doesn't matter whether it's done in JavaScript, CSS, or another way, but it has to be implemented if you want your website to meet the current industry standards. A lot of people are on the move so your website should be responsive! I'm looking forward to practicing CSS-specific ways to create a responsive navigation bar.

How this relates to what I have/will learn in WDD

I researched and learned simple ways to create a reponsive navigation bar with both JavaScript and CSS when I took Designing for Web Standards II. I intend to research and learn more ways of creating a responsive navigation bar with less code.

How I might build this

  1. Create the component as a <section>
  2. Design the navigation bar using CSS such as { display: inline; }
  3. Use media queries to change how the navi bar looks when viewed by certain sizes @media (min-width: 480px) { }
